# 三国霸业

一款使用 Python + Pygame 开发的三国题材策略游戏，复刻经典三国霸业游戏体验。

## 游戏特色

- **经典玩法**: 复刻经典三国霸业游戏的核心玩法
- **精美画面**: 精心设计的游戏界面和视觉效果，包含粒子特效和动画
- **策略深度**: 包含内政、外交、战斗等多种策略元素
- **中文支持**: 完整的简体中文界面

## 游戏内容

### 势力系统
- 魏国 - 曹操
- 蜀国 - 刘备
- 吴国 - 孙权
- 其他势力（袁绍、袁术、吕布、董卓、马腾、公孙瓒等）

### 武将系统
- 40+ 三国名将
- 武力、智力、统率、政治四维属性
- 独特的武将技能系统

### 城市系统
- 15+ 三国时期城市
- 人口、金钱、粮草、兵员管理
- 建筑升级系统

### 战斗系统
- 即时战斗模式
- 步兵、骑兵、弓兵协同作战
- 战斗特效和动画
- 自动战斗选项

### 音效系统
- 完整的音效支持
- 场景背景音乐
- 战斗音效和UI音效

### AI系统
- 智能AI决策
- 势力性格差异
- 自动发展和扩张

### 外交系统
- 结盟、议和、宣战
- 送礼、贸易
- 势力关系管理

### 存档系统
- 5个存档槽位
- 随时保存和读取游戏进度

### 教程系统
- 新手引导教程
- 游戏操作提示
- 可跳过的教程流程

### 日志系统
- 完整的游戏运行日志
- 方便问题定位和分析

### 设置系统
- 音量控制（主音量、音乐、音效）
- 实时预览音量变化

## 安装说明

### 从源码运行

1. 克隆仓库
```bash
git clone https://github.com/ganecheng-ai/three-kingdoms-glm5.git
cd three-kingdoms-glm5
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行游戏
```bash
python main.py
```

### 下载发布版本

前往 [Releases](https://github.com/ganecheng-ai/three-kingdoms-glm5/releases) 页面下载对应平台的可执行文件。

## 系统要求

- Python 3.10+
- Pygame 2.5.0+

## 操作说明

- **鼠标左键**: 选择、确认
- **鼠标右键**: 拖拽地图
- **鼠标滚轮**: 缩放地图
- **空格键**: 暂停战斗
- **ESC键**: 返回上一级菜单

## 项目结构

```

├── main.py              # 游戏入口
├── config.py            # 配置文件
├── game/                # 游戏核心模块
├── scenes/              # 场景模块

├── systems/             # 游戏系统

├── data/                # 游戏数据
```
